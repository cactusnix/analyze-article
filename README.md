## analyze article

在阅读大量文章的时候，我发现很多文章都是标题党或者内容极其枯燥无聊，口水文，所以想要开发一个分析文章内容进行筛选的工具。

希望工具可以做到将文章内容分词，统计词频，通过词频的判断来确定文章主要涉及的是哪些方面。
